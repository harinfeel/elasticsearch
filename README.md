<h1 align="center"> Install Elasticsearch 6.8.5 </h1>

### Requirement

```yaml
OS : CentOS 7
Ansible : 2.9.2
Python: 3.x
```

### Install 

```sh
ansible-playbook -i hosts inst_es.yml -t elasticsearch
```
